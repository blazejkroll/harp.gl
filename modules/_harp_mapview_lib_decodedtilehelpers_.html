applyTechniqueToMaterial

  • applyTechniqueToMaterial(technique: Technique, material: THREE.Material, level?: undefined | number, skipExtraProps?: string[]): void
    • Defined in harp-mapview/lib/DecodedTileHelpers.ts:360

    Apply generic technique parameters to material.

    Skips non-material Technique props:

    • BaseTechnique props,
    • name which is used as discriminator for technique types,
    • props starting with _
    • props found skipExtraProps

    THREE.Color properties are supported.

    Parameters

    • technique: Technique

      technique from where params are copied

    • material: THREE.Material

      target material

    • Optional level: undefined | number

      optional, tile zoom level for zoom-level dependent props

    • Optional skipExtraProps: string[]

      optional, skipped props

    Returns void
