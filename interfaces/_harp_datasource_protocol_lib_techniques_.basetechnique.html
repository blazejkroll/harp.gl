Optional fadeFar

fadeFar: undefined | number
  • Defined in harp-datasource-protocol/lib/Techniques.ts:91

Distance to the camera (0.0 = camera position, 1.0 = farPlane) at which the object has zero opacity and stops fading out. A value of <= 0.0 disables fading.

Optional fadeNear

fadeNear: undefined | number
  • Defined in harp-datasource-protocol/lib/Techniques.ts:85

Distance to the camera (0.0 = camera position, 1.0 = farPlane) at which the object start fading out (opacity decreases).

renderOrderBiasGroup: undefined | string
  • Defined in harp-datasource-protocol/lib/Techniques.ts:61
